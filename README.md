# 📌 Overview

The **AI Cyber Threat Detector Framework** is an AI-powered cybersecurity application that leverages Machine Learning algorithms to identify potential cyber threats from user-provided security-related data.

The framework provides instant predictions through an intuitive Streamlit web interface, helping security professionals, students, and researchers analyze suspicious activities quickly and efficiently.

# 🧠 How It Works

1. User enters cyber-related information.
2. The data is preprocessed.
3. The trained Machine Learning model analyzes the input.
4. AI predicts whether the activity is:
   - Safe
   - Suspicious
   - Potential Threat
5. Results are displayed instantly with relevant insights.

# 💻 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AI-Cyber-Threat-Detector.git
```

Move into the project directory

```bash
cd AI-Cyber-Threat-Detector
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
